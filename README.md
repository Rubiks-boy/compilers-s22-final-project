# CS132 Hasty 

This extension provides language support for Hasty, Tasty, and NormTarget toy languages. These are languages used in Compilers class at Harvey Mudd College.

## Features

Hasty
- Syntax hilighting
- Goto definition
- Hover to show type
- Format Document
- Error Diagnosis
- Basic Autocomplete
- Folding

Tasty
- Syntax Hilighting

NormTarget
- Syntax Hilighting
